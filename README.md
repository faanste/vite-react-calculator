# Vite React Calculator

Jednostavna web aplikacija kalkulator napravljena koristeći React, Vite i TypeScript.

## Funkcionalnosti

- Sabiranje
- Oduzimanje
- Množenje
- Dijeljenje

Aplikacija omogućava unos dva broja i prikaz rezultata odabrane operacije.

## Tehnologije

- React
- Vite
- TypeScript

## Pokretanje projekta

```bash
npm install
npm run dev
